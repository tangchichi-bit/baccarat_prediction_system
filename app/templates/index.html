系統說明
赫茲公式計算方法：
  1. 優勢牌值計算：
    • 1、4、5、7、8 → +2
    • 2、3 → -3
    • 6、9 → -5
    • 10、J、Q、K → 0
  2. 閒家頻率計算（點數總和的個位數）：
    • 7、8、9 → +2
    • 0、1、3、4 → +1
    • 2、5、6 → -5
  3. 莊家頻率計算（點數總和的個位數）：
    • 7、8、9 → +3
    • 0、1、3、4、6 → +2
    • 2、5 → -5
  4. 赫茲結果 = 優勢牌值 + 閒家頻率 + 莊家頻率
  5. 結果為負數時，下一局選擇莊家；結果為正數時，下一局選擇閒家
AI深度學習：

系統使用LSTM神經網絡分析歷史數據，學習百家樂的出牌規律，預測下一局結果。

綜合預測模式：

結合AI預測和赫茲公式結果，根據AI置信度和兩者一致性，給出最終預測建議。
